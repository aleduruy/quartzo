# Touchable
The **Touchable** component must be used to wrap any existing content that will trigger an action when it is touched. 

### Example
<Touchable onPress={() => alert('I was clicked!')} style={{ border: '1px solid #eee', padding: '10px' }}>
  <Text>Click here to see the Touchable in action.</Text>
</Touchable>
