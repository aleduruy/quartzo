# Touchable
O **Touchable** deve ser usado para envolver componentes que possuam uma ação ao toque, como Views, imagens, Badges e outros componentes.

>Caso o conteúdo seja apenas texto e leve o usuário para outra página, considere utilizar o componente [Link](/component-link).

## Default
<Touchable ripple={false} onPress={() => alert('Ação ao clicar!')} style={{ border: '1px solid #eee', padding: '10px' }}>
  <Text>Clique aqui para ver o touchable</Text>
</Touchable>
