# Text
The **Text** component must be used to wrap any existing Text on the Layout. On the web environment it can render a couple of different HTML tags, wich will keep the page semantics.

### Examples
<View>
  <Text>Hello! I'm a default Text example.</Text>
</View>

<View>
  <Text weight={300}>I'm a Text with weight 300.</Text>
</View>
<View>
  <Text weight={400}>I'm a Text with weight 400.</Text>
</View>
<View>
  <Text weight={700}>I'm a Text with weight 700.</Text>
</View>
<View>
  <Text weight={900}>I'm a Text with weight 900.</Text>
</View>

<View>
  <Text italic={true}>I'm a Text with italic style.</Text>
</View>

<View>
  <Text underline={true}>I'm a Text with underline style.</Text>
</View>

### Semantics
Possible tags to be used:
- `label`
- `strong`
- `address`
- `time`
- `span`
