# So, what is Quartzo?

Quartzo is a Javascript Framework that provides normalised [React](https://github.com/facebook/react) and [React-native](https://github.com/facebook/react-native) components, wich means that every component here can be reused on both web and native applications.
All of them follows the concepts of [Atomic Design](http://bradfrost.com/blog/post/atomic-web-design/), a methodology that can offer a good Design System hierarchy for the page that will use them.

The major principles of Quartzo includes the reuse of code, once it's written just one time, the performance of the applications, the visual patterns that a design system can offer, and the accessibility of the page in the case of the web environment.

## Entry Point

Toda aplicação que usar o Cross Theme precisa ter uma tag `ThemeProvider` em volta de todo conteúdo, que será responsável por passar a brand e a version que será utilizada.

``` javascript
import React, { Component } from 'react';
import { ThemeProvider, View, Text } from 'Quartzo';

class Example extends Component {
  render() {
    return (
      <ThemeProvider theme={{ name: "dark" }}>
        <View>
          <Text>Example of a Component.</Text>
        </View>
      </ThemeProvider>
    );
  };
}
```

## Running on web

Primeiro veja se possui as dependências necessárias ao Cross-Theme:
- Yarn
- Node 8+
- Xcode e Android Studio para desenvolver em native [(Como configurar)](https://facebook.github.io/react-native/docs/getting-started.html).

Logo após clonar [o projeto do GitLab](http://gitlab.b2w/team/b2c-product-page/cross-theme), instale as dependências:

```
yarn
```

Para rodar a documentação local utilize:

```
yarn docz:dev
```

Para visualizar no native:

```
yarn dev:native
```
